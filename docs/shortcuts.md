## Kit パネル（Event 行選択中）

| キー | 動作 |
|------|-----|
| `↑` / `↓` | 前後の Event 行に移動 |
| `←` / `→` | Intensity を ±5% 調整 |
| `Space` | 選択中 Event を再生 / 停止トグル |
| `Enter` | 選択中 Event の編集モーダルを開く |
| `Esc` | モーダルを閉じる |

## 編集モーダル

| キー | 動作 |
|------|-----|
| `Tab` | 次のフィールドにフォーカス |
| `Enter` | 確定して閉じる |
| `Esc` | キャンセルして閉じる |

## Library パネル

| キー | 動作 |
|------|-----|
| `↑` / `↓` | クリップ間移動 |
| `Space` | プレビュー再生 / 停止 |
| `Enter` | アクティブ Kit に追加 |
| ドラッグ（☰ ハンドル） | Kit 行の クリップフィールドにドロップ |

## マウス操作の補足

- **wheel**: 各リストの内部スクロール
- **drag-and-drop**: Library クリップ → Kit Event 行にドロップ、またはKit 内 Event 行の並び替え

## 設定

ショートカットはカスタマイズ不可です。`?` キーで開くポップオーバーに最新の一覧が表示されます。
