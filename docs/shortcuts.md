## Kit Manager

Kit パネル内で行を選択している状態で次のキーが効きます。

| キー | 動作 |
|------|-----|
| `↑` / `↓` | 前後の Event 行に移動 |
| `←` / `→` | 前後の Kit 切替（Kit 一覧パネル） |
| `Space` | 選択中 Event を再生（プレビュー） |
| `Space`（再生中） | 一時停止トグル |
| `Enter` | 選択中 Event の編集モーダルを開く |
| `Esc` | モーダルを閉じる |
| `?` | ショートカットヘルプ ポップオーバーを表示 |
| `Ctrl + S` | Save & Deploy |

## 編集モーダル

| キー | 動作 |
|------|-----|
| `Tab` | 次のフィールドにフォーカス |
| `Enter` | 確定して閉じる |
| `Esc` | キャンセルして閉じる |

## Library パネル

| キー | 動作 |
|------|-----|
| `↑` / `↓` | クリップ間移動 |
| `Space` | プレビュー再生 |
| ドラッグ | Event 行の クリップフィールドにドロップ |

## マウス操作の補足

- **wheel**: 各リストの内部スクロール（バブルアップしない）
- **double-click**: クリップ名で実ファイル名を rename（モーダル内 Name blur で確定）
- **drag-and-drop**: クリップ → Event クリップフィールド、または Event 行の並び替え

## 設定

ショートカットは現在カスタマイズ不可です。`?` キーで開くポップオーバーに最新の一覧が出ます。
